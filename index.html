Instructions

Click this window to hide the instructions

By default you will see a base map of Fobney meadow.
You can zoom in and out and drag the map around for a better view.
Features like the pylons are numbered and benches are highlighted to show where they are.

Show location

If you have GPS in your browser (i.e. you're using a phone) then this will show where you are on the map. Click Turn off location to remove your dot

Creating and modifying data

Create features

You can record points, lines, circles or polygons by choosing from the drop-down below next to create and clicking create.
The cursor will change to a circle to show where you are creating a feature.

  • Point - click where you want a point.
  • Line - click the first point of the line, click any intermediate points and double click the last point to finish.
  • Polygon - click all the points of the polygon and double click or right click the last one that joins it up.
  • Circle - click the centre of the circle and click again once it's the size you want.

Modify features

Click create then allow the cursor to snap to the point or vertex on the line or polygon you want to move. Whilst holding down the mouse button move the point to where you want it and release.

Select features

Click select then select a feature. Hold down shift to select more than one.

Name features

When a feature is selected you may name it in the Feature name box. Be careful! If more than one feature is selected they will all get that one name!

Delete features

Once you have selected some features, hit delete to delete those selected.

Download data

Allows you to download the data for your own use or to send to me (huggie@earth.li) to add to the base map

Adding Balsam sightings

Show himalayan-balsam data

Shows the base himalayan balsam data layer

Show this data

Shows the data in the text box to the left of the button on the map instead

Hide data

Hides all the data layers data

Submit data

Lets you submit data you have created. First load the current base layer, then create some data over your own (see Create above) then when you are happy click Submit data.
You will be asked for a comment that should be descriptive (no spaces but a-z A-Z 0-9 and - are allowed) and indicates if this is additional balsam or successful removals for instance. Then hit submit and it should be available in the drop down to look at.
